About Django Pixel

Django Pixel is a modern, production-ready Bootstrap theme designed for Django applications. It provides a comprehensive set of UI components, utilities, and layouts.

Key Features

  • 50+ Custom Components
  • Bootstrap 5 Based
  • Font Awesome Icons
  • Responsive Design
  • SCSS Customization
  • JavaScript Plugins Included

Installation

pip install django-theme-pixel

Utilities & Helpers

Spacing Utilities

Margin and padding utilities using Bootstrap's spacing scale (0-5):

  • m-{0-5} - Margin on all sides
  • mt-{0-5} - Margin top
  • mb-{0-5} - Margin bottom
  • p-{0-5} - Padding on all sides
  • pt-{0-5} - Padding top
  • pb-{0-5} - Padding bottom
Example: mt-3 mb-4 p-2
